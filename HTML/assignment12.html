    5 loại vòng lặp trong Javascript

  1. Vòng lặp while

  2. Vòng lặp do..while

  3. Vòng lặp for

  4. Vòng lặp for..in

  5. Vòng lặp for..of

Vòng lặp while

Ví dụ:
 var i = 1;
 while(i <= 5) { document.write("<p>Số i là: " + i + "</p>"); i++; }
Kết quả in ra
 Số i là: 1
 Số i là: 2
 Số i là: 3
 Số i là: 4
 Số i là: 5

Vòng lặp do...while

Ví dụ:
 var i = 1;
 do { document.write("<p>Số i là: " + i + "</p>"); i++; }
 while(i <= 5);
Kết quả in ra
 Số i là: 1
 Số i là: 2
 Số i là: 3
 Số i là: 4
 Số i là: 5
Vòng lặp do...while luôn thực thi một lần ngay cả khi biểu thức điều kiện là false

Vòng lặp for

Ví dụ:
 // Một mảng trong JavaScript
 var hoaQua= ["Táo", "Chuối", "Xoài", "Cam", "Lựu"];
  // Lặp qua các phần tử trong mảng
  for(var i = 0; i < hoaQua.length; i++) {
  document.write("<p>" + hoaQua[i] + "</p>");
 }
Kết quả in ra
 Táo
 Chuối
 Xoài
 Cam
 Lựu
Vòng lặp for đặc biệt hữu ích cho việc lặp qua một mảng trong JavaScript.

Vòng lặp for...in

Ví dụ:
 // Một đối tượng
 var person = {"Tên": "Doanh", "Họ": "Vi", "Tuổi": "36"};
 // Lặp qua tất cả các thuộc tính
 for(var x in person) {
 document.write("<p>" + x + " = " + person[x] + "</p>");
 }
Kết quả in ra
 Tên : Doanh
 Họ : Vi
 Tuổi : 36
Vòng lặp for...in thích hợp lấy thuộc tính của đối tượng (giá trị phần tử trong mảng), nhưng không dùng cho mảng (đối tượng) mà mảng đó quan trọng thứ tự chỉ mục

Vòng lặp for...of

Ví dụ:
 // Tạo một mảng chuẩn bị để lặp
 let letters = ["a", "b", "c", "d", "e", "f"];
 for(let letter of letters) {
 console.log(letter); // a,b,c,d,e,f
 }
 // Lặp qua một chuỗi
 let greet = "Hello World!";
 for(let character of greet) {
 console.log(character); // H,e,l,l,o, ,W,o,r,l,d,!
 }
Vòng lặp for...of không dùng cho đối tượng (mảng trong mảng), muốn lấy thuộc tính của đối tượng có thể dùng for...in
